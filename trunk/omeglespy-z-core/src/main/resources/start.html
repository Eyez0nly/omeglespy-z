Welcome to OmegleSpy-Z

OmegleSpy-Z is an opensource application that allows you to 'spy' on peoples conversations on Omegle. You can also send text as either user, block messages, disconnect users, and even swap a user without the other knowing.

For help, press F1.

Visit OmegleSpy-Z's homepage at http://code.google.com/p/omeglespy-z/

If you have any issues, glitches, or feature requests please submit them on https://code.google.com/p/omeglespy-z/issues/list.
