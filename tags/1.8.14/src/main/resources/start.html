Welcome to OmegleSpy-Z

OmegleSpy-Z is a freeware and opensource application that allows you to 'spy' on peoples conversations on Omegle. You can also send text as either user, block messages, disconnect users, and even swap a user without the other knowing. Some future plans include support for spying on video mode chats as well. Visit OmegleSpy-Z's homepage at http://code.google.com/p/omeglespy-z/

If you have any issues, glitches, or feature requests please submit them on the webpage.
