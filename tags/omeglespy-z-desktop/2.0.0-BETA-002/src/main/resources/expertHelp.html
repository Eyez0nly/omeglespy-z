OmegleSpy-Z Expert Help

Common keyboard shortcuts available in both normal and expert modes:

Ctrl+N Toggle conversation state. If you have started a conversation, this chord ends the conversation. Otherwise, this chord starts a new conversation.
Ctrl+B Toggle whether or not strangers are blocked from sending messages to each other. The application begins with strangers not being blocked.
Ctrl+F Toggle whether or not filtering is enabled. By default, filtering is not enabled. If filtering is enabled, strangers who send a message that matches any of the strings in conf/filter.txt will be automatically disconnected.
Ctrl+S Toggle auto-scroll. Auto-scroll is enabled by default. When auto-scroll is disabled, new messages will not cause the chat history window to automatically scroll to the last message submitted.
Ctrl+U Clears the chat history window of all messages.
Ctrl+L Shows the Chat History Viewer. The Chat History Viewer contains a read out of all chats in which you have engaged during this session. The list will not be saved if you exit the application.
Ctrl+Q Exits the application. An alternative to Alt+F4 (which does the same thing).
F1 Toggles the display of this help message.
Ctrl+E Toggles expert mode. In expert mode, you have more keyboard shortcuts available to you, but you also only get one text box into which to type secret messages. See below for more details.
